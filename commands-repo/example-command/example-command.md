# Example Command

This is an example custom slash command to demonstrate the commands system. Copy or symlink it into `commands/` to activate:

```bash
cd ~/aide
ln -s ../commands-repo/example-command/example-command.md commands/example-command.md
```

Once activated, use it in Claude Code with:

```
/example-command
```
